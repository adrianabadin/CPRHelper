Create `AclsTracker/ViewModels/AuthViewModel.cs` using CommunityToolkit.Mvvm:

1. **Observable properties:**
   - `string Email`, `string Password` (for login form)
   - `string RegisterEmail`, `string RegisterPassword`, `string Nombre`, `string Apellido`, `string Telefono` (for register form)
   - `bool IsLoggedIn`, `string UserDisplayName`, `string? UserAvatarUrl` (auth state)
   - `bool IsBusy` (loading indicator)
   - `UserProfile? CurrentProfile` (profile page)
   - `string? ErrorMessage` (for inline error display)
   - `bool IsGoogleAvailable` â€” true when `DeviceInfo.Platform != DevicePlatform.WinUI`
   - `bool IsAppleAvailable` â€” true when platform supports it (iOS or Android for web flow)

2. **Commands (RelayCommand):**
   - `SignInWithEmailCommand` â†’ calls `_authService.SignInWithEmailAsync(Email, Password)`, on success navigates back (modal pop), on failure sets ErrorMessage and shows toast
   - `SignUpCommand` â†’ calls `_authService.SignUpWithEmailAsync(RegisterEmail, RegisterPassword, Nombre, Apellido, Telefono)`, on success shows toast "Revisa tu email para verificar tu cuenta" and navigates to LoginPage, on failure shows error
   - `SignInWithGoogleCommand` â†’ calls `_authService.SignInWithGoogleAsync()`, on success navigates back
   - `SignInWithAppleCommand` â†’ calls `_authService.SignInWithAppleAsync()`, on success navigates back
   - `SignOutCommand` â†’ calls `_authService.SignOutAsync()`, updates IsLoggedIn/UserDisplayName/UserAvatarUrl
   - `ResetPasswordCommand` â†’ calls `_authService.ResetPasswordAsync(Email)`, shows toast "Email de recuperacion enviado"
   - `LoadProfileCommand` â†’ calls `_authService.GetProfileAsync()`, populates CurrentProfile fields
   - `SaveProfileCommand` â†’ calls `_authService.UpdateProfileAsync(CurrentProfile)`, shows success toast
   - `UploadAvatarCommand` â†’ uses `MediaPicker.Default.PickPhotoAsync()`, then `_authService.UploadAvatarAsync(filePath)`, updates UserAvatarUrl
   - `NavigateToLoginCommand` â†’ `Shell.Current.GoToAsync("LoginPage")`
   - `NavigateToRegisterCommand` â†’ `Shell.Current.GoToAsync("RegisterPage")`
   - `NavigateToProfileCommand` â†’ `Shell.Current.GoToAsync("ProfilePage")`

3. **Constructor:** Inject `IAuthService`. Subscribe to `AuthStateChanged` to update IsLoggedIn, UserDisplayName, UserAvatarUrl. Check initial auth state.

4. **Error handling:** All commands wrapped in try/catch. Use `CommunityToolkit.Maui.Alerts.Toast.Make(message).Show()` for user feedback. Set `IsBusy = true` before async ops, `false` in finally block.

5. **All UI text in Spanish:** "Iniciar sesion", "Registrarse", "Cerrar sesion", "Email enviado", "Error al iniciar sesion", etc.
