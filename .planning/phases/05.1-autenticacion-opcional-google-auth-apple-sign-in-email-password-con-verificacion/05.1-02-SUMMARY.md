**AuthService with Supabase SDK for email/password, Google OAuth, Apple Sign-In, and session persistence - all IAuthService methods implemented and integrated into MAUI DI**

## Accomplishments
- Implemented AuthService.cs with all IAuthService methods using Supabase SDK
- Email login calls Supabase.Auth.SignIn, signup creates user and upserts profile
- Google OAuth uses PKCE flow with ProviderAuthState returned by SignIn(Provider)
- Apple Sign-In uses native iOS ASAuthorizationController with PKCE fallback
- Logout clears session and fires AuthStateChanged event
- Password reset calls Supabase.Auth.ResetPasswordForEmail
- Profile CRUD via Supabase.From<UserProfile>().Upsert/Get
- Avatar upload to "avatars" storage bucket with public URL
- UserProfile model updated to inherit from BaseModel for ORM compatibility
- Supabase client registered as singleton with SetPersistence before InitializeAsync
- App.xaml.cs restores session on startup using fire-and-forget pattern
