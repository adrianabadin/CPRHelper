**MainPage.xaml changes:**

1. **Remove the EventLogPanel block** — Delete the entire Block 6 comment and control:
   ```xml
   <!-- BLOCK 6: EventLog below the fold (scrollable within page scroll) -->
   <controls:EventLogPanel BindingContext="{Binding EventRecording}" />
   ```
   Do NOT delete EventLogPanel.xaml or EventLogPanel.xaml.cs — the control itself stays in the project for the Historial tab.

2. **Add floating tab bar toggle button** — Inside the root `<Grid>` (same level as ScrollView and NotificationBanner), add a floating circular button at bottom-right:
   ```xml
   <!-- Tab bar toggle button — floating, bottom-right -->
   <Button x:Name="TabBarToggle"
           Text="&#x2630;"
           WidthRequest="42" HeightRequest="42"
           CornerRadius="21"
           BackgroundColor="#1A73B8"
           TextColor="White"
           FontSize="18"
           VerticalOptions="End" HorizontalOptions="End"
           Margin="0,0,14,14"
           ZIndex="5"
           Clicked="OnToggleTabBarTapped" />
   ```
   NOTE: Using hex color `#1A73B8` here for now. Plan 01 runs in parallel and will replace it with `{StaticResource AccentClinical}` — if Plan 01 completes first, use the resource. If not, the gap closure or manual update will catch it. The important thing is the button exists and works.

**MainPage.xaml.cs changes:**

1. **Add a private field** to track tab bar visibility state:
   ```csharp
   private bool _isTabBarVisible = false;
   ```

2. **Override OnAppearing** to hide the tab bar by default:
   ```csharp
   protected override void OnAppearing()
   {
       base.OnAppearing();
       Shell.SetTabBarIsVisible(this, false);
       _isTabBarVisible = false;
   }
   ```
   Use `Shell.SetTabBarIsVisible(this, bool)` — the static method from code-behind (NOT XAML binding, per research pitfall #1).

3. **Add toggle event handler:**
   ```csharp
   private void OnToggleTabBarTapped(object sender, EventArgs e)
   {
       _isTabBarVisible = !_isTabBarVisible;
       Shell.SetTabBarIsVisible(this, _isTabBarVisible);
       TabBarToggle.Text = _isTabBarVisible ? "\u2715" : "\u2630"; // X when visible, hamburger when hidden
   }
   ```

Do NOT modify any ViewModel. Do NOT change AppShell.xaml tab definitions. The tab bar toggle is purely a view-layer concern handled in code-behind.
