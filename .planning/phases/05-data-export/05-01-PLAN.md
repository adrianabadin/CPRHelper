   The PDF must contain these 6 sections per user's locked decision:

   **Section 1 — Header:**
   - Title: "Resumen de Resucitación ACLS"
   - Patient: `{PatientLastName}, {PatientName}`
   - DNI: `{PatientDNI}`
   - Date/time: `{SessionStartTime:dd/MM/yyyy HH:mm}`
   - Duration: calculated from SessionEndTime - SessionStartTime, formatted as "Xm Xs"
   
   **Section 2 — Resumen de Ritmo (Rhythm Summary):**
   - Filter events where EventType == "RhythmChange"
   - Table with 2 columns: Tiempo (ElapsedSinceStart as mm:ss) | Ritmo (Description)
   - If no rhythm events: show "Sin cambios de ritmo registrados"

   **Section 3 — Medicamentos Administrados:**
   - Filter events where EventType == "Medication"
   - Table with 2 columns: Medicación (Description) | Tiempo (ElapsedSinceStart as mm:ss)
   - If no medication events: show "Sin medicamentos registrados"

   **Section 4 — Causas Reversibles (H's y T's):**
   - Filter events where EventType == "HsTs"
   - Table with 2 columns: Causa (Description) | Tiempo (ElapsedSinceStart as mm:ss)
   - If no HsTs events: show "Sin causas reversibles registradas"

   **Section 5 — Compresiones y Ciclos:**
   - Filter events where EventType == "CprCycle"
   - List or table showing compression cycles with relative times
   - Include Defibrillation events here too (EventType == "Defibrillation") with times
   - If neither: show "Sin registro de compresiones/desfibrilaciones"

   **Section 6 — Footer:**
   - Centered text: "Documento generado por ACLS Tracker — Protocolo AHA 2020"
   - Small font size (8pt), grey color

   Implementation details:
   - Use `Document.Create(container => { container.Page(page => { ... }); }).GeneratePdf()` returning byte[]
   - Page size A4, margin 1.5cm
   - Default text style 10pt
   - Section titles: bold, 12pt, blue color, bottom border
   - Tables: header row with grey background, alternating row padding
   - Wrap CPU-bound GeneratePdf() in `Task.Run` to avoid blocking UI thread
   - Write byte[] to file in `FileSystem.Current.CacheDirectory` with naming: `ACLS_{PatientLastName}_{SessionStartTime:yyyyMMdd_HHmm}.pdf`
   - Return the full file path

   IMPORTANT: Do NOT call QuestPDF.Settings.License here — that belongs in MauiProgram.cs (Task 2).
   Use `using QuestPDF.Fluent;`, `using QuestPDF.Helpers;`, `using QuestPDF.Infrastructure;` at the top.

   }

2. Create `AclsTracker/Services/Export/CsvExportService.cs` implementing ICsvExportService.

   Per user's locked decision, the CSV must have these 6 columns:
   - `Timestamp` — ISO 8601 format (evt.Timestamp.ToString("o"))
   - `Tiempo_relativo` — mm:ss format (ElapsedSinceStart)
   - `Tipo_evento` — EventType string
   - `Descripcion` — Description string
   - `Detalles` — Details string (or empty)
   - `Ritmo_actual` — ONLY populated for RhythmChange events (use Description for those), empty for all other events

   Implementation details:
   - Use `StreamWriter` with `new UTF8Encoding(true)` for BOM (critical for Spanish Excel compatibility)
   - Comma delimiter
   - Escape fields containing commas, quotes, or newlines with standard CSV quoting: wrap in double quotes, double any internal quotes
   - File naming: `ACLS_{PatientLastName}_{SessionStartTime:yyyyMMdd_HHmm}.csv`
   - Write to `FileSystem.Current.CacheDirectory`
   - Order events by Timestamp ascending
   - Return full file path

   Example header row: `Timestamp,Tiempo_relativo,Tipo_evento,Descripcion,Detalles,Ritmo_actual`
   Example data row: `2026-03-30T14:30:00.000Z,00:00,RhythmChange,FV,,FV`
   Example data row: `2026-03-30T14:32:00.000Z,02:00,Medication,Adrenalina 1mg,,`
