# Plan 03: Pause Icon Toggle + Amiodarone After 2 Defibrillations

## Goal

Two independent UI/protocol improvements:
1. **Pause icon toggle**: TimerCard pause button shows ⏸ (pause) when running, ▶ (play/resume) when paused
2. **Amiodarone condition**: Amiodarone suggestion only appears AFTER 2 or more defibrillations have occurred (AHA ACLS 2020: amiodarone is for refractory VF/pVT after shocks)

## Architecture

- **Task 1 (UI)**: Add `IsPaused` binding to Button.Text in TimerCard.xaml, with a ValueConverter to return ⏸ or ▶
- **Task 2 (Protocol)**: Add `_defibrillationCount` to MainViewModel, increment on each Defibrilar(), check count≥2 before suggesting amiodarone

## Task 1: Pause Icon Toggle

**Files:**
- Modify: `AclsTracker/Controls/TimerCard.xaml:67-76`

**Step 1: Add IValueConverter for pause/resume icon**

Create `AclsTracker/Converters/PauseIconConverter.cs`:

```csharp
using System.Globalization;
using Microsoft.Maui.Controls;

namespace AclsTracker.Converters;

public class PauseIconConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        return value is bool isPaused && isPaused ? "▶" : "⏸";
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        throw new NotImplementedException();
    }
}
```

**Step 2: Register converter in App.xaml.cs**

Find `Resources` dictionary and add:

```csharp
converter:PauseIconConverter />
```

**Step 3: Apply converter to TimerCard pause button**

In `AclsTracker/Controls/TimerCard.xaml`, change the pause Button:

```xml
<!-- Pause/Resume button — visible when ShowPauseButton=true on this control -->
<Button Grid.Row="2" Grid.Column="1"
        WidthRequest="36"
        HeightRequest="36"
        FontSize="16"
        Padding="0"
        BackgroundColor="Transparent"
        TextColor="#FF6F00"
        Text="{Binding IsPaused, Converter={StaticResource PauseIconConverter}}"
        Command="{Binding PauseCommand, Source={x:Reference TimerCardControl}}"
        IsVisible="{Binding ShowPauseButton, Source={x:Reference TimerCardControl}}" />
```

Note: Removed hardcoded `Text="⏸"` — now bound to IsPaused.

## Task 2: Amiodarone After 2 Defibrillations

**Files:**
- Modify: `AclsTracker/ViewModels/MainViewModel.cs:23-34, 390-406`

**Step 1: Add _defibrillationCount field and property**

In `MainViewModel.cs`, add after line 28 (`private bool _hasCompletedCode;`):

```csharp
/// <summary>
/// Tracks number of defibrillations in current code session.
/// Amiodarone is indicated only after 2+ shocks (refractory VF/pVT per AHA ACLS 2020).
/// </summary>
private int _defibrillationCount;
```

**Step 2: Increment count in Defibrilar() method**

Modify `Defibrilar()` (around line 401-406):

```csharp
[RelayCommand]
private void Defibrilar()
{
    _defibrillationCount++;
    DefibrillationTriggered?.Invoke();
    EventRecording.LogCustomEventCommand.Execute("Defibrilación realizada");
}
```

**Step 3: Update GetSuggestedDrug() for amiodarone condition**

Find the `GetSuggestedDrug()` method. The current amiodarone condition checks `_amiodaronaDoseCount < 2`. We need to add `_defibrillationCount >= 2` as a prerequisite.

Current relevant code:
```csharp
// TV/FV, cycleCount >= 1
bool amiodaronaDue = (_lastAmiodaronaTime == null || (DateTime.Now - _lastAmiodaronaTime.Value).TotalMinutes >= 3)
                     && _amiodaronaDoseCount < 2;
```

Change to:
```csharp
// TV/FV, cycleCount >= 1, AND at least 2 defibrillations have occurred
bool amiodaronaDue = _defibrillationCount >= 2
                     && (_lastAmiodaronaTime == null || (DateTime.Now - _lastAmiodaronaTime.Value).TotalMinutes >= 3)
                     && _amiodaronaDoseCount < 2;
```
