    **1. Create ISessionSyncService** (`Services/Sync/ISessionSyncService.cs` — NEW):
    ```csharp
    using AclsTracker.Models;

    namespace AclsTracker.Services.Sync;

    /// <summary>
    /// Orchestrates session sync between local SQLite and Supabase.
    /// Listens to AuthStateChanged for automatic claim/download/cleanup triggers.
    /// </summary>
    public interface ISessionSyncService
    {
        /// <summary>Upload a session (and its events) to Supabase. Queues on failure.</summary>
        Task UploadSessionAsync(Session session, List<EventRecord> events);

        /// <summary>Delete all sessions for the given user from local SQLite only.</summary>
        Task DeleteLocalUserSessionsAsync(string userId);

        /// <summary>Event fired when sync operations complete (for UI refresh).</summary>
        event EventHandler? SyncCompleted;
    }
    ```

    **2. Create SessionSyncService** (`Services/Sync/SessionSyncService.cs` — NEW):

    Constructor dependencies: `IAuthService`, `ISessionRepository`, `Supabase.Client` (registered as singleton).

    Key design decisions:
    - Subscribe to `_authService.AuthStateChanged` in constructor → `OnAuthStateChanged` handler
    - Use in-memory `Queue<Func<Task>>` for failed uploads with exponential backoff retry timer
    - All sync operations are fire-and-forget (non-blocking) — wrap in try/catch with Debug.WriteLine
    - Order of operations on login: (1) claim orphans → (2) download user sessions → (3) retry pending uploads

    **UploadSessionAsync(Session session, List\<EventRecord\> events):**
    1. Map Session → SessionSupabase (copy all fields including UserId)
    2. Map List\<EventRecord\> → List\<EventSupabase\> (use `evt.ToEntity(session.Id)` pattern to get entity fields, then map to EventSupabase)
    3. Insert session: `await _supabase.From<SessionSupabase>().Insert(sessionModel);`
    4. Insert events in batches of 100: `await _supabase.From<EventSupabase>().Insert(batch);`
    5. On exception: enqueue retry delegate, start retry timer if not running
    6. Debug.WriteLine for success/failure logging

    **OnAuthStateChanged handler:**
    ```csharp
    private async void OnAuthStateChanged(object? sender, bool isLoggedIn)
    {
        if (isLoggedIn)
        {
            var userId = _authService.CurrentUserId;
            if (!string.IsNullOrEmpty(userId))
            {
                await HandleLoginSyncAsync(userId);
            }
        }
        else
        {
            // Note: cleanup is triggered from AuthViewModel which has the userId
            // This handler does NOT handle cleanup (userId is gone after logout)
        }
    }
    ```

    **HandleLoginSyncAsync(string userId):**
    ```csharp
    private async Task HandleLoginSyncAsync(string userId)
    {
        try
        {
            // Step 1: Claim all orphan sessions (mark UserId + upload)
            await ClaimOrphanSessionsAsync(userId);

            // Step 2: Download ALL user sessions from Supabase
            await DownloadUserSessionsAsync(userId);

            // Step 3: Retry any previously failed uploads
            await ProcessRetryQueueAsync();

            // Notify UI to refresh
            SyncCompleted?.Invoke(this, EventArgs.Empty);
        }
        catch (Exception ex)
        {
            Debug.WriteLine($"[SessionSyncService] Login sync failed: {ex.Message}");
        }
    }
    ```

    **ClaimOrphanSessionsAsync(string userId):**
    1. Get orphans: `var orphans = await _sessionRepo.GetOrphanSessionsAsync();`
    2. For each orphan:
       a. Call `UpdateSessionUserIdAsync(orphan.Id, userId)`
       b. Set `orphan.UserId = userId`
       c. Get events via `GetSessionEventsAsync(orphan.Id)`
       d. Call `UploadSessionAsync(orphan, events)` (will map and insert into Supabase)
    3. Wrap in try/catch per session — one failure shouldn't block others

    **DownloadUserSessionsAsync(string userId):**
    1. Query Supabase: `var result = await _supabase.From<SessionSupabase>().Where(x => x.UserId == userId).Get();`
    2. For each remote session:
       a. Check if `GetSessionAsync(remote.Id)` already exists → skip if so
       b. Map SessionSupabase → Session (SQLite model): copy all fields, set UserId
       c. Query events: `var remoteEvents = await _supabase.From<EventSupabase>().Where(x => x.SessionId == remote.Id).Get();`
       d. Map EventSupabase → EventRecordEntity: copy all fields
       e. Call `InsertDownloadedSessionAsync(session, eventEntities)`
    3. Wrap in try/catch per session

    **DeleteLocalUserSessionsAsync(string userId):**
    1. Simply delegate: `await _sessionRepo.DeleteByUserIdAsync(userId);`
    2. Debug.WriteLine for logging

    **Retry queue mechanism:**
    - `private readonly Queue<Func<Task>> _uploadQueue = new();`
    - `private IDispatcherTimer? _retryTimer;`
    - On upload failure: enqueue the upload delegate
    - Start a retry timer (30 seconds, then exponential: 30s, 60s, 120s, capped at 5 min)
    - `ProcessRetryQueueAsync()`: dequeue and execute each, remove on success, re-enqueue on failure
    - Max retries per item: 5 (track via counter in queue items)

    **IMPORTANT patterns to follow:**
    - Use `System.Diagnostics.Debug.WriteLine` for all logging (no ILogger dependency needed)
    - Use `.ConfigureAwait(false)` on all async calls (library code, no UI context needed)
    - Event mapping from EventRecord → EventSupabase: access `ElapsedSinceStart.Ticks` for `ElapsedTicks`, and use the entity mapping from `EventRecord.ToEntity()` pattern
    - Supabase `Insert()` can take a `List<T>` for bulk insert — use for events in batches of 100

    **CRITICAL: Mapping EventRecord → EventSupabase**
    EventRecord has `ElapsedSinceStart` (TimeSpan) but EventSupabase needs `ElapsedTicks` (long). The existing `EventRecordEntity` stores `ElapsedTicks` — look at how `EventRecord.ToEntity()` works. The sync service receives `List<EventRecord>`, needs to convert to `EventSupabase`. Map like this:
    ```csharp
    var eventModel = new EventSupabase
    {
        Id = evt.Id,
        SessionId = sessionId,
        Timestamp = evt.Timestamp,
        ElapsedTicks = evt.ElapsedSinceStart.Ticks,
        EventType = evt.EventType,
        Description = evt.Description,
        Details = evt.Details
    };
    ```
