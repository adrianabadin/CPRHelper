   d. Change `StopCode` method from `void` to `async Task`:
      - `[RelayCommand]` attribute natively supports async Task methods in CommunityToolkit.Mvvm 8.x
      - First, stop timers and recording (existing logic stays):
        ```csharp
        _pulseCheckTimer?.Stop();
        _pulseCheckTimer = null;
        _chargingWarningTimer?.Stop();
        _chargingWarningTimer = null;
        Timer.StopSessionCommand.Execute(null);
        Timer.ResetAllCommand.Execute(null);
        EventRecording.StopRecordingCommand.Execute(null);
        ```
      - Then show patient data popup and save:
        ```csharp
        // Capture session data BEFORE resetting state
        var sessionStartTime = EventRecording.Events.LastOrDefault()?.Timestamp ?? DateTime.Now;
        // Actually use EventLogService.SessionStartTime — access via the service
        // Since EventRecordingViewModel wraps IEventLogService, capture events now
        var eventsToSave = EventRecording.Events.ToList();

        // Show patient data popup
        var popup = new PatientDataPopup();
        var result = (PatientDataPopup.PatientDataResult?)await Application.Current!.MainPage!
            .ShowPopupAsync(popup);

        if (result is not null)
        {
            var session = new Session
            {
                Id = Guid.NewGuid().ToString(),
                PatientName = result.Nombre,
                PatientLastName = result.Apellido,
                PatientDNI = result.DNI,
                SessionStartTime = eventsToSave.Count > 0
                    ? eventsToSave.Min(e => e.Timestamp)
                    : DateTime.Now,
                SessionEndTime = DateTime.Now,
                CreatedAt = DateTime.Now
            };
            await _sessionRepository.SaveSessionAsync(session, eventsToSave);
        }
        ```

      - Finally, reset session state (existing logic stays):
        ```csharp
        _cycleCount = 0;
        _amiodaronaDoseCount = 0;
        IsAmiodaronaEnabled = false;
        _adrenalinaBannerFired = false;
        _amiodaronaBannerFired = false;
        _lastAdrenalinaTime = null;
        _lastAmiodaronaTime = null;
        IsAdrenalinaSuggested = false;
        IsAmiodaronaSuggested = false;
        ```

   e. NOTE: Do NOT call ClearEvents() here. The existing code doesn't call it, and events accumulate in memory. The next StartCode will overwrite the session state anyway. The saved data is already in SQLite.

   f. NOTE about SessionStartTime: The IEventLogService has `SessionStartTime` property. Since EventRecordingViewModel holds the IEventLogService, we need access to it. Options:
      - Option A: Add a public property `DateTime? SessionStartTime => _eventLogService.SessionStartTime;` to EventRecordingViewModel
      - Option B: Capture min timestamp from events list
      Use Option A — it's cleaner. Add this one-line property to EventRecordingViewModel:
      ```csharp
      public DateTime? SessionStartTime => _eventLogService.SessionStartTime;
      ```
      Then in StopCode: `SessionStartTime = EventRecording.SessionStartTime ?? DateTime.Now`
