```csharp
private async void OnPulseCheckDue(object? sender, EventArgs e)
{
    _pulseCheckTimer?.Stop();

    // Build medication suggestion lines based on ACLS 2020 protocol
    var suggestions = new List<string>();

    // ADRENALINA — ACLS: 1mg IV/IO cada 3-5 minutos para todos los ritmos de paro
    // Timer[3] (Adrenalina) resets when user presses ADRENALINA, so IsOverThreshold
    // means >4 min since last dose — correct protocol timing check.
    // If adrenaline was given in the previous cycle (2 min ago), timer shows ~2 min
    // which is < 4 min threshold → no suggestion. Timer naturally handles cycle awareness.
    if (Timer.Timers[3].IsOverThreshold)
    {
        suggestions.Add("¿Hora de Adrenalina?");
    }

    // AMIODARONA — ACLS: para FV/TV refractaria, máx 2 dosis
    // 1ra dosis: 300mg tras 2do shock
    // 2da dosis: 150mg tras dosis adicional
    // Suggest only if: timer >4 min AND rhythm is shockable AND max doses not reached
    if (Timer.Timers[4].IsOverThreshold &&
        EventRecording.CurrentRhythm is CardiacRhythm.TV or CardiacRhythm.FV &&
        _amiodaronaDoseCount < 2)
    {
        string doseHint = _amiodaronaDoseCount == 0
            ? "¿Hora de Amiodarona? (1ra dosis: 300mg)"
            : "¿Hora de Amiodarona? (2da dosis: 150mg)";
        suggestions.Add(doseHint);
    }

    // Build message with suggestions (if any)
    string message = "Han pasado 2 minutos.\nConstate pulso y ritmo.\nAdministre 2 ventilaciones.";

    if (suggestions.Count > 0)
    {
        message += "\n\n" + string.Join("\n", suggestions);
    }

    string? result = await Application.Current!.MainPage!
        .DisplayAlert("Check de Pulso", message, "DEFIBRILAR", "CONTINUAR", "CANCELAR");

    if (result == "DEFIBRILAR")
    {
        EventRecording.LogCustomEventCommand.Execute("Defibrilación realizada");
    }
    else if (result == "CONTINUAR" || result == "CANCELAR")
    {
        Timer.PauseCompressions();
        Timer.StartPulseCheckTimer();
    }
}
```

## ACLS Protocol Logic Summary

**Adrenalina** (timer-based, naturally cycle-aware):
- Timer resets on ADRENALINA press → tracks real time since last dose
- IsOverThreshold (true when >4 min) = sufficient interval per ACLS (every 3-5 min)
- If given in previous cycle (~2 min ago): timer shows ~2 min < 4 min → no suggestion ✓
- If NOT given for >4 min: timer > 4 min → suggests ✓
- Works for ALL arrest rhythms (adrenaline is universal in ACLS)

**Amiodarona** (protocol-aware with dose tracking):
- Only for shockable rhythms (TV/FV) — amiodarone not indicated for asystole/PEA
- _amiodaronaDoseCount tracks cumulative doses: 0, 1, 2 (max per ACLS)
- After 2 doses: no more suggestions (protocol max reached)
- Suggestion includes dose number and mg amount for clinical clarity
- Timer ensures sufficient interval between doses

Pattern reference: IsAmiodaronaEnabled (line 35) uses same rhythm check pattern. Amiodarona() and Adrenalina() methods (lines 99-111) are the commands that reset the timers and log events.
