**IEventLogService.cs** — Service contract:
```csharp
namespace AclsTracker.Services.EventLog;
public interface IEventLogService
{
    ObservableCollection<EventRecord> Events { get; }
    DateTime? SessionStartTime { get; }
    void StartSession();         // Records session start, sets SessionStartTime
    void EndSession();           // Records session end
    EventRecord LogEvent(string eventType, string description, string? details = null);
    void ClearEvents();          // Reset for new session
}
```

**EventLogService.cs** — Implementation:
- `Events`: ObservableCollection<EventRecord>, newest first (insert at index 0)
- `SessionStartTime`: Set on StartSession(), used to calculate ElapsedSinceStart for each event
- `LogEvent()`: Creates EventRecord with `DateTime.Now` (millisecond precision per REGI-02), calculates `ElapsedSinceStart` from `SessionStartTime`, generates GUID for Id, inserts at position 0 in Events collection
- `StartSession()`: Sets SessionStartTime = DateTime.Now, logs "Inicio de código" event
- `EndSession()`: Logs "Fin de código" event
- `ClearEvents()`: Clears collection and resets SessionStartTime
- Thread-safe: use lock around Events modifications (same pattern as TimerService)

**EventRecordingViewModel.cs** — ViewModel exposing all Phase 2 functionality:
- Constructor injection: `IEventLogService`, `ITimerService` (to read elapsed times)
- Properties:
  - `CurrentRhythm` (CardiacRhythm, ObservableProperty) — current selected rhythm, default Ninguno
  - `Events` => `_eventLogService.Events` (pass-through for binding)
  - `HsAndTsItems` (ObservableCollection<HsAndTsItem>) — initialized with standard ACLS H's and T's
  - `IsSessionActive` (bool, ObservableProperty)

- H's and T's initialization (in constructor, Spanish names):
  H's: Hipovolemia, Hipoxia, Hidrogeniones (Acidosis), Hipo/Hiperpotasemia, Hipotermia
  T's: Neumotórax a Tensión, Taponamiento Cardíaco, Tóxicos, Trombosis Pulmonar, Trombosis Coronaria

- Commands (all [RelayCommand]):
  - `SelectRhythm(CardiacRhythm rhythm)` — Sets CurrentRhythm, logs event "Ritmo: {rhythm display name}" via EventLogService. Map enum values to Spanish display names (e.g., FibrilacionVentricular → "FV/TV")
  - `ToggleHsAndTsItem(HsAndTsItem item)` — Toggles IsChecked, sets CheckedAt = DateTime.Now, logs event "H&T considerado: {item.Name}" or "H&T descartado: {item.Name}" depending on IsDismissed state
  - `DismissHsAndTsItem(HsAndTsItem item)` — Sets IsDismissed = true, IsChecked = true, CheckedAt = DateTime.Now, logs event "H&T descartado: {item.Name}"
  - `LogCustomEvent(string description)` — Logs a free-text event via EventLogService
  - `StartRecording()` — Calls EventLogService.StartSession(), sets IsSessionActive = true
  - `StopRecording()` — Calls EventLogService.EndSession(), sets IsSessionActive = false
